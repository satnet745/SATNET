## 🔑 KEY FEATURES (Unified App)

### SERVAL MESH (Communication)
✅ Voice calling (direct P2P + multi-hop)  
✅ Video calling (H.264)  
✅ Text messaging (MeshMS)  
✅ File sharing (Rhizome)  
✅ Offline mesh network  
✅ No SIM card required  

### BITCOIN (Self-Custody Wallet)
✅ BIP39 (12-word recovery)  
✅ BIP32 (hierarchical keys)  
✅ Private key management  
✅ PIN protection (AES-256-GCM)  
✅ Cloud backup (encrypted)  
✅ Offline signing  

### VOUCHER SYSTEM
✅ Agent generation  
✅ QR code encoding/scanning  
✅ Cryptographic verification  
✅ Single-use enforcement  
✅ Time-limited expiry  
✅ Offline capability

✅ Merchant invoices

✅ Real-time rates
