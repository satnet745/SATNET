Current SATNET help summary

This help bundle has been refreshed for the current SATNET app structure visible in this repository.
The main screen now combines classic communication features with SATNET financial tools in a separate SATNET card.
Current user-facing capabilities documented here include:
  • mesh calling, messaging, contacts, and Rhizome file sharing
  • SATNET Maps with local-only temporary markers, encrypted saved bookmarks, and optional Rhizome bookmark exchange
  • Bitcoin wallet setup and wallet access
  • role-based SATNET flows for agent vouchers, voucher redemption, verifier review, and build-dependent merchant Lightning tools
  • updated Bluetooth permission handling on modern Android
For the full repository release notes, open CURRENT-RELEASE.md when you have internet access.
