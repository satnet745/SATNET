SATNET Connect

The Connect screen manages the radios and network modes SATNET uses to find peers and carry traffic. Communication features become much more useful when SATNET services are enabled and at least one reachable peer is available.

The Enable Services checkbox starts or stops the SATNET mesh services. When enabled, the app can announce itself, discover peers, relay traffic, and synchronize mesh-related services. This can increase battery use while active.

Below that toggle, SATNET shows available network controls and connection options supported by the device.

Wi-Fi

The Wi-Fi entry opens the normal Android Wi-Fi settings flow or toggles Wi-Fi participation depending on the device and Android version.

Use this when you want to join an existing local network that other SATNET users can also reach.

Bluetooth

On supported devices, Bluetooth can help with nearby peer discovery and local mesh links.

On Android 12 and newer, SATNET may request Bluetooth runtime permissions before it can scan, connect, or request discoverable mode. If you deny those permissions, Bluetooth mesh features remain limited until you grant them again.

Portable Wi-Fi Hotspot

Hotspot mode lets your device act as a local access point so nearby users can join through you.

Turning on a hotspot usually disables normal Wi-Fi client mode while it is active. If your mobile data plan is enabled, nearby devices may use it, which can cost you money.

Ad Hoc Mesh

Ad hoc Wi-Fi is a legacy and highly device-specific mode. It often requires root access and may attempt risky low-level Wi-Fi changes on older hardware.

This mode is kept mainly for specialist testing and historical compatibility. Most users should prefer ordinary Wi-Fi, hotspot, and Bluetooth-based flows instead.
