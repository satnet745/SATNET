Installing SATNET on your phone

To allow the software to install, you have to go to: Settings -> Applications, and allow installation of software from Unknown Sources. If you do not enable this, you will be prompted to do so.
To install SATNET, please download the Android package:
Download
You can follow the download of the package file in the Navigation Bar of your Android phone.
When it is finished downloading, click on it & follow the instructions to install it.
