═══════════════════════════════════════════════════════════════════════════
 IF BUILD FAILS
═══════════════════════════════════════════════════════════════════════════

1. Check NDK Installation:
   Android Studio → Tools → SDK Manager → SDK Tools tab
   Ensure "NDK (Side by side)" version 21.4.7075529 is installed

2. Clean and Rebuild:
   gradlew.bat clean
   gradlew.bat assembleDebug --info

3. Check the Error:
   - "missing file" → Source file doesn't exist
   - "undefined reference" → Linking issue (check module order)
   - "undefined module" → Dependency not built yet (check include order)

4. Stop Gradle Daemon (if hanging):
   gradlew.bat --stop
